GitHub repo name:
ray5-lightburn-bridge

GitHub description:
A LightBurn-to-Ray5 Wi-Fi bridge that provides GRBL Ethernet/TCP compatibility, HTTP control, and touchscreen-friendly SD upload mode for Longer Ray5 lasers.

Short project summary:
This project bridges LightBurn's GRBL Ethernet/TCP mode to the Longer Ray5's HTTP-based Wi-Fi interface. It supports live control, protocol probing, and an SD spool mode that uploads Ray5-screen-friendly offline job files such as `longer_001.gc.gz` for manual selection on the machine.
