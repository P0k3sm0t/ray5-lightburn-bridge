GitHub repo name:
ray5-lightburn-bridge

GitHub description:
A local TCP-to-Ray5 bridge that lets LightBurn talk to a Longer Ray5 over Wi-Fi using GRBL Ethernet/TCP, with HTTP bridging and experimental SD spool/upload mode.

Short project summary:
This project bridges LightBurn's GRBL Ethernet/TCP mode to the Longer Ray5's HTTP-based Wi-Fi interface. It supports direct command forwarding, protocol probing, and an experimental SD spool mode that uploads jobs to the Ray5 and starts them from onboard storage to reduce live-stream jitter.
